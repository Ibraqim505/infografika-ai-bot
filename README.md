# AI Infografika SaaS Bot

Production-ready Telegram bot:
- Webhook (Render-ready)
- OpenAI image generation
- Logging system
- Media storage
- .env configuration

## DEPLOY TO RENDER

1. Upload to GitHub
2. Create Web Service on Render
3. Build Command:
   pip install -r requirements.txt
4. Start Command:
   python bot.py
5. Add Environment Variables in Render:
   BOT_TOKEN
   OPENAI_API_KEY
   BASE_WEBHOOK_URL
